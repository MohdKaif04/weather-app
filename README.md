# 🌦️ Weather App

A responsive web application that provides real-time weather information for any city using a weather API. The app allows users to search for a location and view current weather conditions such as temperature, humidity, and wind speed.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Real-time temperature display
* 💨 Wind speed and humidity details
* 📱 Responsive UI for all devices
* ⚡ Fast API-based data fetching

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **API:** OpenWeatherMap API
* **Tools:** Fetch API, DOM Manipulation

---

## ⚙️ How It Works

1. User enters a city name
2. App sends request to weather API
3. API returns JSON data
4. Data is parsed and displayed on UI
